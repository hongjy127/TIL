### 익명함수

- Android 필수기능

- 함수에 이름이 없음, 변수를 할당해서 사용
- 람다함수 형태를 주로 사용

```kotlin
{
    매개변수 -> 함수본체
}
```

```kotlin
val 변수명: (매개변수 목록) -> 리턴타입 = 익명함수(또는 함수명)
// 매개변수가 1개면 이름대신 it 키워드 사용
```
